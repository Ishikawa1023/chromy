# chromy

chromy is a controller for headless mode chrome.  
chromy means headless in japanese language.

## Installation

```bash
npm i chromy
```

## Usage

```js
>> <amp-img src="./spec/data/rect.jpg" width="100" height="100" layout="responsive"></amp-img>
```
